Sound Comparisons – Offline Version

  Version: June 21nd 2018

The offline version without sound files needs roughly 17GB of free hard drive space.

Please follow these steps to get the offline version working on your computer:

  • download this ZIP-archive (81MB) and decompress it at any destination and you get a new folder called sndComp_offline
  • if you do not already have the folder mapnik somewhere on your computer download this ZIP-archive (3.3GB) to the folder sndComp_offline and decompress it [It will take a long time!] into the folder sndComp_offline and you get a new folder called mapnik
  • download the sound files per study (under construction)
    • Europe
    • Germanic
    • Romance
    • Englishes
    • Celtic
    • Slavic
    • Andes
    • Mapudungun
    • Brazil (123MB)
    • Malakula (1.2GB)
    and decompress the downloaded archive and copy the folders into the sound folder (overwrite already existing sound files). If the folder sound does not yet exist create it simply.
  • after downloading all files the folder sndComp_offline should now have the following structure: [bold indicates a folder] 
    	css
	data
	img
	index.html
	js
	LICENSE
	mapnik
	README.md
	sound
  • open the file index.html inside the folder sndComp_offline with any browser
    except with Internet Explorer or Microsoft Edge since these browsers do not support the underlying technology
